This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Getting Started

```bash
npm install
npm run build
npm run start
```

Open [http://localhost:3000](http://localhost:3000) with your browser. Click the button or link and you'll see the delay in adding those search params to the URL as the parent page is `force-dynamic` with a 2000ms blocking promise.
